# Search Documentation

Use the search box at the top of the page to find information across all Nextended documentation. The search functionality is built into the theme and provides instant results as you type.

## Search Tips

- **Use specific terms**: Search for method names, class names, or concepts
- **Partial matching**: The search supports partial word matching
- **Multiple keywords**: Use multiple words to narrow down results
- **Case insensitive**: Search is not case-sensitive
- **Preview context**: Search results show context around the matched term
